## How Does it Work?
  * **Step 1:** User click on the weather icon in the middle of screen that will then proceed to generate a random weather condition                  and a Marvel Hero that relates to the current weather. 

<img src="https://media.giphy.com/media/kHZyTYGuwzNXMeeLkZ/giphy.gif" width="100%" height="100%" />

* **Step 2:** While on the Marvel Hero page, the user can explore an overview of the hero, Action shots, An image andd quote gallery, and also meet the real life hero.  

<img src="https://media.giphy.com/media/chQkjQuDFPQMgrRTN2/giphy.gif" width="100%" height="100%" />


****Note : There is a navbar on the right handside with the list of hero for the user to view. Although some hero may not pop up depending on the weather conditions and the current season.
